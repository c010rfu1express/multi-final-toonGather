## 멀티잇 백엔드 개발 25회차 파이널 프로젝트 [툰게더 toonGather]
'toonGather'는 'toon'(웹툰)과 'gather'(모으다)의 결합으로, 웹툰 문화의 모든 요소를 한데 모아 새로운 가치를 창출하는 플랫폼으로 기획한 프로젝트입니다.

## 💫 팀 소개
소비자 및 창작자 모두를 위한 웹툰 통합 플랫폼을 지향합니다.
- 팀장 및 프로젝트 매니징 총괄
  - 최서윤
- 팀원
  - 박재형 
  - 박서영
  - 김동현
  - 김희철
  - 강현구
